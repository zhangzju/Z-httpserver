# Z-httpserver

Use different golang web-framework to create RESTful web services.

# How to use

* beego: inspired by tornado and Sinatra

get beego first:

```shell
go get gtihub.com/astaxie/beego
```
Run the entry file:
```shell
go build -o beego-test ./beego/index.go
```

Start the binary file:
```shell
./beego-test
```
